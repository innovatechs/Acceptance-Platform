Not sure which sign-in to use?

You need to sign in with the same username and password every time you use My Service Canada Account.

  • When you first registered for a My Service Canada Account, you selected a particular sign-in method.
  • You might have created a new GCKey user id and password just for this account, and written it down or saved it in a Password manager.
  • Or when you registered, you may have decided to use your existing financial institution username and password rather than create a new GCKey for Service Canada.
  • My Service Canada Account tied that particular sign-in username and password with your tax and benefit information. Use that same method every time you sign in.
